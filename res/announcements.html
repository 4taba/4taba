- (2018-04-14) Site updated to Github version.
This is still not quite v1.0 yet, but it's is in a usable state. Almost every aspect of the site was recoded including the database tables which were completely migrated to a different format to get rid of some of the "cruft" the codebase picked up from past feature experiments.
