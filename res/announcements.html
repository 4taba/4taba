Announcements



- (2018-05-07) Site updated to Github version.
This marks the beginning of 4taba as a real imageboard. Prior to this it has mostly been treated as a sandbox.

The site has been updated to version 1.0 and placed on track with the Github version, which means much faster and more reliable updates in the future (instead of the previous shotgun approach in which all updates from the past few months would be added to the site all at once). The entire codebase was overhauled with almost every aspect of the site being recoded including the database tables which were migrated to a completely different format to get rid of some of the "cruft" the codebase picked up from past feature experiments. Some bug regressions may be possible, and a few (lesser used) features still need to be reimplemented, such as the thread watcher.

Other noteworthy changes include:
- Standardized post formatting (three > signs for crossthread/crossboard linking and [spoiler] and [code] instead of [sp] and [cb]). This way people don't have to remember special nuances of 4taba and post formatting will work with what they're already familiar with from other imageboards.
- Catalog search works again
- User post/thread deletion works again
- Mod interface works again
- Mods can view reported posts from within the site (previously reports just got dumped in a text file on the server)
